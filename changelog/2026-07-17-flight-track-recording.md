### Added

- Flight track recording: since this fleet has no DJI Dock, there is no after-the-fact flight history to import from the cloud platform for a manually flown mission. Instead, the mission editor's DJI Cloud panel can now record a device's live GPS trace while flying (start before takeoff, stop after landing) and draw the actually-flown path on the map next to the planned route for any past recording tied to the open mission.
